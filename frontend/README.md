# Frontend — Chat UI

Интерфейс для общения с ИИ-агентом по подбору гитар.

## Технологии

| Компонент | Технология |
|-----------|-----------|
| Сборка | Vite |
| UI-фреймворк | React 18 |
| Язык | TypeScript |

## Запуск

```bash
# Установка зависимостей
npm install

# Запуск dev-сервера
npm run dev

# Сборка для продакшена
npm run build

# Предпросмотр сборки
npm run preview
```

Интерфейс: http://localhost:5173

## Структура

```
src/
├── features/
│   └── chat/              # Модуль чата
│       ├── components/    # React-компоненты
│       │   ├── Chat.tsx
│       │   ├── MessageList.tsx
│       │   ├── Message.tsx
│       │   └── InputForm.tsx
│       ├── hooks/         # Кастомные хуки
│       ├── types.ts       # Типы данных
│       └── index.ts       # Экспорты модуля
├── shared/
│   └── components/        # Переиспользуемые компоненты
├── App.tsx               # Корневой компонент
└── main.tsx              # Точка входа
```

## Архитектура

Проект организован по принципу **feature-based**:
- Каждая фича — самодостаточна
- Общий код выносится в `shared/`
- Из `features/` не импортируем внутрь друг друга

## API

Общение с backend через `/api/` (FastAPI):
- Backend: `http://localhost:8000`
- Frontend: `http://localhost:5173`

## Компоненты

### Chat
Главный компонент чата. Управляет состоянием и рендерингом.

### MessageList
Список сообщений. Показывает empty state или список сообщений.

### MessageItem
Одно сообщение (пузырь). Разные стили для user/agent.

### InputForm
Форма ввода. Поле + кнопка отправки. Отправка по Enter.

## Состояния интерфейса

- **Empty state** — приветствие с примером запроса
- **Загрузка** — индикатор после отправки запроса
- **Ошибка** — сообщение об ошибке подключения
