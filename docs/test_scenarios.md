# Тестовые сценарии ИИ-агента

В этом документе описаны базовые сценарии (test cases) для проверки логики ИИ-агента по подбору гитар. В будущем они станут основой для автоматизированного тестирования.

## Сценарий 1: Прямой подбор с четкими параметрами
* **Запрос пользователя:** "Привет! Ищу яркую электрогитару для блюза и фанка, бюджет до 1000 баксов."
* **Ожидаемое поведение (Режим 1 - Поиск):**
  * Бюджет: $<1000
  * Из-за "яркая" -> Синглы
  * "блюз и фанк" -> Stratocaster / Telecaster
* **Действие:** Вызов скрипта с JSON-запросом:
  ```json
  {
    "search_queries": [
      "Fender Player Stratocaster",
      "Squier Classic Vibe Telecaster"
    ],
    "price_max": 1000
  }
  ```

## Сценарий 2: Нехватка данных
* **Запрос пользователя:** "Хочу купить гитару для метала."
* **Ожидаемое поведение (Режим 2 - Консультация):**
  * Извлеченных данных недостаточно для точного поиска (нет бюджета, конкретного типа).
* **Действие:** Задать уточняющие вопросы (какой бюджет? 6, 7 или 8 струн? форма корпуса?).

## Сценарий 3: Теоретический вопрос
* **Запрос пользователя:** "Чем отличаются синглы от хамбакеров?"
* **Ожидаемое поведение (Режим 2 - Консультация):**
  * Это вопрос об устройстве инструмента.
* **Действие:** Объяснить разницу в уровне шума (фон) и характере звука (яркий/прозрачный vs плотный/громкий). **Не вызывать функцию поиска.**

## Сценарий 4: Выход за рамки компетенции (Out of Scope)
* **Запрос пользователя:** "Как сгенерировать картинку в Midjourney?"
* **Ожидаемое поведение:**
  * Запрос не относится к гитарам.
* **Действие:** Вежливый отказ ("Извините, я специализируюсь только на подборе гитар и звуке...").

## Сценарий 5: Акустическая гитара
* **Запрос пользователя:** "Нужна гитара для костра петь песни, недорогая, до 200 долларов."
* **Ожидаемое поведение (Режим 1 - Поиск):**
  * Бюджет: $<200
  * Тип: Акустическая (для костра)
* **Действие:** Вызов скрипта с JSON-запросом:
  ```json
  {
    "search_queries": [
      "Yamaha F310",
      "Fender CD-60",
      "Epiphone DR-100"
    ],
    "price_max": 200
  }
  ```

## Сценарий 6: Противоречивый запрос
* **Запрос пользователя:** "Хочу акустическую гитару с флойд роузом и активными звукоснимателями EMG."
* **Ожидаемое поведение (Режим 2 - Консультация):**
  * Акустических гитар с такой комплектацией не выпускают.
* **Действие:** Объяснить пользователю техническое противоречие и предложить либо электрогитару для тяжелой музыки, либо электроакустику с пьезодатчиком.
