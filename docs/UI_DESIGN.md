# Chat UI Design

## Обзор

Интерфейс чата для общения с ИИ-агентом по подбору гитар. Пользователь описывает желаемый звук → агент возвращает рекомендации.

## Компоненты интерфейса

### 1. Header
- Название приложения: "🎸 Guitar Agent"
- Фиксированная высота: 60px
- Цвет фона: тёмно-синий/серый

### 2. Область чата (Message Area)
- Занимает всё доступное пространство между header и input
- Вертикальная прокрутка при переполнении
- Сообщения выравниваются:
  - Пользователь — справа (или слева с акцентом)
  - Агент — слева

### 3. Сообщение (Message Bubble)
- Максимальная ширина: 70% экрана
- Скруглённые углы
- Разные цвета для user/agent
- Время отправки в углу пузыря
- Иконка отправителя (👤 / 🤖)

### 4. Форма ввода (Input Form)
- Текстовое поле: 80% ширины
- Кнопка отправки: 20% ширины (иконка ➤ или текст "Отправить")
- Фиксированная высота: 60px
- Блокировка кнопки при пустом поле

### 5. Индикатор загрузки (Loading Indicator)
- Показывается после отправки запроса
- Текст: "Агент подбирает гитары..."
- Анимация: прогресс-бар или пульсирующий текст

## Состояния интерфейса

### 1. Начальное состояние (Empty State)
```
┌─────────────────────────────────────────────────────────┐
│  🎸 Guitar Agent                                        │
├─────────────────────────────────────────────────────────┤
│                                                         │
│           🎸 Добро пожаловать в Guitar Agent!           │
│                                                         │
│     Опишите желаемый звук и бюджет — я найду           │
│           подходящие варианты на Reverb.                │
│                                                         │
│     Пример: "Хочу телекастер с ярким звуком, до $600"   │
│                                                         │
├─────────────────────────────────────────────────────────┤
│  [Введите ваш запрос...]                    [Отправить] │
└─────────────────────────────────────────────────────────┘
```

### 2. Ввод текста
- Поле активно
- Кнопка отправки активна (не заблокирована)
- Курсор мигает в поле ввода

### 3. Отправка запроса / Загрузка
- Поле ввода блокируется
- Кнопка блокируется
- Показывается индикатор загрузки
- Сообщение пользователя уже отображено

### 4. Отображение ответа
- Сообщение агента добавляется в чат
- Индикатор загрузки скрывается
- Поле ввода разблокируется
- Скролл автоматически прокручивается вниз

### 5. Ошибка (Error State)
```
┌─────────────────────────────────────────────────────────┐
│  ⚠️ Ошибка                                              │
│  Не удалось подключиться к серверу. Попробуйте ещё раз. │
│                                          [Попробовать]  │
└─────────────────────────────────────────────────────────┘
```

## Адаптивность

### Desktop (> 768px)
- Максимальная ширина чата: 800px
- Центрирование на странице

### Mobile (< 768px)
- Чат на всю ширину экрана
- Кнопка отправки — иконка вместо текста
- Уменьшенные отступы

## Взаимодействия

| Действие | Реакция |
|----------|---------|
| Клик в поле ввода | Фокус, показ курсора |
| Ввод текста | Отображение символов |
| Enter (без Shift) | Отправка формы |
| Клик "Отправить" | Отправка формы |
| Отправка | Блокировка UI, показ лоадера |
| Получение ответа | Разблокировка, скролл вниз |
| Ошибка | Показ error-сообщения |
