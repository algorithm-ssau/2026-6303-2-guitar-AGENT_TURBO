# Недельные задачи команды

> Папки создаются по мере необходимости — не заранее.
> Когда заканчивается текущая неделя, создаётся следующая папка.

---

## Структура

Каждая папка `week-XX/` содержит:
- `TODO_<фамилия>.md` — задача каждого активного участника недели

До week-08 команда планировалась на 6 участников. Начиная с week-09 активных участников 5: без `TODO_khasanov.md`.

Каждый TODO = **1 задача → 3 шага → 3 коммита**.

---

## Требования по коммитам (из PROJECT_PLAN.md)

| Уровень | Коммитов в месяц / чел. | Суммарно за проект |
|---------|------------------------|--------------------|
| Минимум | 3 | 54 |
| Для уверенной защиты | 6–10 | 100+ |

---

## Именование веток

Формат: `feature/<фамилия>/<описание>`

Примеры:
- `feature/salnikov/chat-ui`
- `feature/pavlov/llm-prompt`
- `feature/mergaliev/search-params`
- `feature/sidorov/reverb-integration`
- `feature/khasanov/ranking`
- `feature/fokin/consultation-mode`

---

## Как создавать новую неделю

1. Создать папку `week-XX/`
2. Создать файлы `TODO_<фамилия>.md` по числу активных участников недели
3. Каждый TODO: 1 задача, 3 шага, 3 коммита, критерии приёмки и команды проверки
