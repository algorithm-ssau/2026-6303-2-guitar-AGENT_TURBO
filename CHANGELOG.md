- **Фокин** — связка полного пайплайна в `service.py` с status-callback'ами, компонент `ModeBadge` (поиск/консультация), E2E WebSocket тесты
- **Хасанов** — ранжирование с упрощённой формулой (budget 55% + title 45%), компонент `BudgetHint` с цветовыми индикаторами цены, edge-case тесты
- **Сидоров** — расширение `mock_reverb.json` до 15–20 листингов, фильтрация в `search_reverb()`, компонент `SearchStatus` с анимацией загрузки

## Неделя 4 — 30 марта – 5 апреля 2026

> Полировка: real-time статусы, camelCase-маппинг, graceful degradation, история чата

- **Павлов** — переключение `Chat.tsx` с HTTP на WebSocket через `useChat`, улучшение промптов с few-shot примерами, flow-тесты с мок-LLM
- **Мергалиев** — snake_case ↔ camelCase маппинг для API-контракта, компонент `StatusIndicator` для real-time статусов, тесты контракта
- **Сальников** — полная реализация `useChat()` с camelCase-маппингом и auto-reconnect, бэкенд отправляет WS status-обновления, интеграционные тесты
- **Фокин** — подключение `router.py` к реальному пайплайну, graceful degradation при отсутствии сервисов, quickstart-гайд и smoke-тесты
- **Хасанов** — удаление score из output, улучшение минимальной формулы ранжирования, компонент `RelevanceBadge` с позициями, интеграционные тесты
- **Сидоров** — история чата с сессиями (локальная реляционная БД, для разработки — SQLite), сайдбар с управлением сессиями, передача истории в контекст LLM
